## 🚀 Tecnologias Utilizadas

O desenvolvimento deste site priorizou performance, design moderno (glassmorphism, animações fluidas) e código limpo, construído com:

- **HTML5** (Semântico e acessível)
- **CSS3 / Tailwind CSS** (Estilização responsiva e componentizada)
- **JavaScript (Vanilla)** (Lógica de animações, Intersection Observer, GSAP/Efeitos)
- **Iconify** (Ícones vetoriais em tempo de execução)

## 🌟 Estrutura e Funcionalidades

O projeto é uma One-Page Application composta pelas seguintes seções:

- **Hero Section**: Introdução de alto impacto com background em vídeo, orbes animados (mix-blend-mode), e efeito *typewriter* de digitação.
- **Sobre Mim**: Timeline de evolução educacional (Senac, Fatec) e conquistas, além das métricas de experiência na área.
- **Soft Skills**: Um card interativo e gamificado estilo "FUT" (FIFA), que apresenta autoavaliações de habilidades como comunicação, liderança e resolução de problemas.
- **Tecnologias & Ferramentas**: Exposição visual (em formato de "pills" animados) do domínio de diversas ferramentas de Front-end (Vue.js, Flutter), Back-end (PHP, Node.js, Python), Bancos de Dados (MySQL) e outras.
- **Experiência Profissional**: Histórico de projetos e contribuições dentro da TV Fronteira, apresentados através de um design de acordeão fluido.
- **Projetos em Destaque**: Espaço reservado para visualização rica de projetos pontuais (Portal ifronteira, Aplicativos, Automações em Python).
- **Contato**: Links sociais diretos para networking e parcerias.

## 📱 Design e Responsividade

O portfólio adota uma abordagem **Mobile-first**, adaptando de forma orgânica a disposição dos elementos (como a distribuição lado a lado da Hero Section para empilhamento vertical em telas menores).
Foi projetado um modo escuro (Dark Mode) fixo que realça cores neon, gradientes e texturas (ruído, scanlines) gerando uma sensação tecnológica imersiva ("Vibe Coding").

## 🛠 Como rodar localmente

Como o projeto se baseia essencialmente em arquivos estáticos (HTML/CSS/JS), o setup para rodar localmente é direto:

1. Clone o repositório:
   ```bash
   git clone https://github.com/kevenescovedo/nome-do-repositorio.git
   ```
2. Entre na pasta do projeto:
   ```bash
   cd nome-do-repositorio
   ```
3. Inicie um servidor local para garantir o funcionamento correto de módulos ou recursos (como vídeos e scripts).
   * Usando a extensão **Live Server** no VSCode.
   * Usando Python:
     ```bash
     python -m http.server 8000
     ```
4. Acesse em seu navegador via `http://localhost:8000`.
